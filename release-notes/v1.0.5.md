## 更新内容

- 新增 `glb/gltf` 模型预览，双击后使用 F3D 打开独立预览窗口。
- 新增模型格式探测：后缀为 `.gltf` 但内容实际为 `glb` 的文件会自动按 `glb` 处理。
- 新增常见图片预览，支持 `png/jpg/jpeg/bmp/gif/webp`。
- 新增目录缓存：已访问目录再次进入时默认使用缓存，显式点击“刷新当前目录”或按 `F5` 才重新加载。
- 修复应用和预览相关窗口图标，补齐 Windows 可执行文件图标。

## 说明

- `gltf` 预览会先按文件内容判断真实格式，再决定按 `glb` 或 `gltf` 处理。
- 模型预览依赖 `f3d`。
